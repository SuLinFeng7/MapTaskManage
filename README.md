# 工作管理主页

一个基于中国地图的浏览器主页，用于管理不同省份的工作任务和发版计划。

## 功能特点

- 🏠 **浏览器主页样式** - 美观的主页界面，包含快速链接功能
- 🗺️ **交互式中国地图** - 支持缩放和拖拽，点击省份管理任务
- 📋 **任务卡片管理** - 为每个省份添加多个任务卡片
- 📅 **发版时间管理** - 设置任务发版时间，自动标记即将发版的任务
- 📊 **工作安排记录** - 详细记录每个任务的工作安排和计划
- 🏷️ **任务状态管理** - 支持待开始、进行中、已完成、已延期等状态
- 🔗 **禅道链接** - 支持添加禅道任务链接，快速跳转
- 📈 **数据统计** - 显示总任务数、即将发版任务数等统计信息
- 💾 **本地存储** - 数据自动保存到浏览器本地存储

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 使用方法

1. **打开主页** - 启动应用后，你会看到一个浏览器主页样式的界面
2. **快速链接** - 使用顶部快速链接功能添加常用网站
3. **查看统计** - 点击统计卡片查看对应的任务列表
4. **管理任务** - 点击地图上的任意省份，在右侧面板中：
   - 点击"添加任务卡片"按钮创建新任务
   - 填写版本名称、工作安排、发版时间、禅道URL等信息
   - 设置任务状态（待开始/进行中/已完成/已延期）
   - 编辑或删除已有任务卡片
5. **发版提醒** - 7天内即将发版的任务会自动高亮显示，地图省份颜色会根据发版时间渐变
6. **数据持久化** - 所有数据自动保存，刷新页面后数据依然存在

## 任务卡片字段

- **版本名称** - 必填，任务的版本名称
- **工作安排** - 详细的工作计划和安排
- **发版时间** - 任务的预计发版日期
- **状态** - 任务当前状态（待开始/进行中/已完成/已延期）
- **备注** - 其他需要记录的信息
- **禅道 URL** - 禅道任务链接，可在任务卡片中直接跳转

## 技术栈

- React 18
- Vite 4
- ECharts - 地图可视化
- Ant Design - UI 组件库
- 本地存储（localStorage）

## 浏览器支持

支持所有现代浏览器（Chrome、Firefox、Safari、Edge）

1. 将代码推送到 GitHub
2. 访问 [vercel.com](https://vercel.com) 并登录
3. 点击 "New Project" 导入你的仓库
4. 点击 "Deploy" 完成部署
5. 在项目设置中配置自定义域名

项目已包含 `vercel.json` 配置文件，可以直接部署。
